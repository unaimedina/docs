> Apartado para mostrar los proof of concepts de las funcionalidades que se quieren implementar en la aplicación.

# Mail Service
El servicio de mailing está provisto por el aplicativo `SendGrid`.

**Pasos de creación**:
- Crear una cuenta en [SendGrid](https://sendgrid.com/)
- Crear una API Key
- Configurar el servicio de mailing en la aplicación

# Paypal
Para implementar PayPal en la aplicación, hemos usado la API de [PayPal](https://developer.paypal.com/developer/applications/).

**Pasos de creación**:
- Crear una cuenta en [PayPal](https://developer.paypal.com/developer/applications/)
- Crear una aplicación
- Configurar la aplicación en la aplicación

# Chart.js
Para la creación de gráficos en la aplicación, hemos usado la librería [Chart.js](https://www.chartjs.org/).

**Pasos de creación**:
- Instalar la librería
- Crear un gráfico
- Configurar el gráfico en la aplicación
