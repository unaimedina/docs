> Voltic Store es un proyecto realizado por Unai Medina y Arnau Muñoz, dos estudiantes de Desarrollo de Aplicaciones Multiplataforma en el INS Nicolau Copernic, de Terrassa. Este proyecto ha sido creado con el objetivo de facilitar la compra de productos electrónicos a través de una interfaz web.

El proyecto se pondrá en producción para el grupo de desarrollo [Voltic Studios](https://voltic.dev).

<CardGroup cols={2}>
  <Card title="¿Qué es Voltic Store?" icon="question">
    Es una tienda online de productos electrónicos, de manera principal, scripts de FiveM y Minecraft. En ella, los usuarios podrán comprar productos de manera segura y rápida.
  </Card>
  <Card title="¿Porqué hemos elegido este proyecto?" icon="circle-info">
    Debido a que era una necesidad para el grupo de desarrollo, hemos decidido crear una tienda online para facilitar la compra de productos electrónicos.
  </Card>
</CardGroup>
